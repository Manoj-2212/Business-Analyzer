Business Analyzer

This project analyzes business data to generate insights such as sales trends and performance metrics.

🚀 Features

- Data analysis
- Visualization
- Business insights

🛠️ Technologies Used

▶️ How to Run

   git clone https://github.com/Manoj-2212/Business-Analyzer.git

2. Go to project folder:
   cd Business-Analyzer

3. Run the file:
   python main.py
